# YouTube Video Search App 

تطبيق أندرويد يتيح للمستخدمين البحث عن مقاطع الفيديو عبر منصة YouTube وعرض النتائج بشكل منظم وديناميكي. تم بناء هذا المشروع كجزء من متطلبات **الواجب الثاني (Assignment 2)** لمساق **تطوير تطبيقات الهواتف الذكية 2 (Mobile App Development 2)**.

##  الميزات البرمجية (Features)
- **البحث الفوري:** إدخال كلمات مفتاحية وجلب النتائج مباشرة باستخدام **YouTube Data API v3**.
- **عرض احترافي:** استخدام `RecyclerView` لعرض تفاصيل الفيديو (العنوان، الوصف، اسم القناة، تاريخ النشر، والصورة المصغرة).
- **كفاءة عالية:** معالجة الصور وتحميلها بسلاسة وكاش ممتاز باستخدام مكتبة `Glide`.
- **اتصال مستقر:** إدارة طلبات الشبكة في الخلفية عبر مكتبتي `Retrofit 2` و `OkHttp`.
- **تجربة مستخدم تفاعلية (UI Behavior):** - إظهار مؤشر تحميل `ProgressBar` أثناء جلب البيانات.
  - معالجة حالات الخطأ (المدخلات الفارغة، انقطاع الإنترنت، أو عدم وجود نتائج).
  - مسح النتائج تلقائياً عند تفريغ حقل البحث.
  - تشغيل الفيديو مباشرة في تطبيق يوتيوب أو المتصفح عند الضغط عليه.

#  التقنيات المستخدمة (Tech Stack)
- **لغة البرمجة:** Java 
- **البيئة التطويرية:** Android Studio
- **المكتبات الخارجية:**
  - Retrofit (للاتصال بالسيرفر وجلب الـ JSON).
  - Gson Converter (لتحويل البيانات تلقائياً لكلاسات جافا).
  - Glide (لتحميل الصور المصغرة).

##  إعداد وتشغيل المشروع (Setup & Run)
1. قم بعمل **Clone** أو تحميل للمستودع (Repository) على جهازك.
2. افتح المشروع باستخدام **Android Studio**.
3. تأكد من مزامنة ملفات الجرادل بالضغط على **Sync Project with Gradle Files**.
4. (اختياري) يمكنك إضافة مفتاح الـ API الخاص بك في متغير `API_KEY` داخل كلاس `MainActivity.java`.
5. قم بتوصيل هاتفك أو تشغيل المحاكي (Emulator) واضغط على **Run**
